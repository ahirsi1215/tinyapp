## Dependencies

- Node.js
- Express
- EJS
- bcryptjs
- cookie-session

## Getting Started

- To install all the dependencies (use the `npm install` command).
- Run the development web server (using the `node express_server.js` command).
- In the terminal (type in `http://localhost:8080/`) to access the web page.
